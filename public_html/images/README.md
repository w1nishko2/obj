# Изображения для инструкций

Эта папка содержит изображения для модального окна с инструкциями, которое показывается пользователю после создания проекта.

## Необходимые изображения:

### 1. tutorial-delete.png
**Описание:** Демонстрация процесса удаления элемента через долгое нажатие
- Показать палец, который удерживает элемент (задачу/этап)
- Можно показать визуальную обратную связь (анимация, индикатор прогресса)
- Рекомендуемый размер: 400-600px по ширине

### 2. tutorial-edit.png
**Описание:** Демонстрация доступа к настройкам задачи/этапа
- Показать как открыть настройки элемента (клик по задаче/этапу)
- Можно показать интерфейс с открытыми настройками
- Рекомендуемый размер: 400-600px по ширине

## Примечания:
- Изображения должны быть в формате PNG с прозрачным фоном (если возможно)
- Используйте скриншоты реального интерфейса приложения
- Добавьте стрелки или выделения для лучшего понимания
- Оптимизируйте размер файлов для быстрой загрузки

## Как добавить изображения:
1. Создайте скриншоты нужных действий в приложении
2. Сохраните их с указанными именами файлов в эту папку
3. Изображения автоматически появятся в модальном окне
