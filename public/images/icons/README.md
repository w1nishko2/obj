# Инструкция по созданию иконок для PWA

Для правильной работы PWA необходимо создать иконки следующих размеров:
- 72x72
- 96x96
- 128x128
- 144x144
- 152x152
- 192x192
- 384x384
- 512x512

## Вариант 1: Использование онлайн-генератора

1. Перейдите на https://www.pwabuilder.com/imageGenerator
2. Загрузите ваш логотип (минимум 512x512px)
3. Скачайте сгенерированные иконки
4. Поместите их в папку `public/images/icons/`

## Вариант 2: Создание вручную

Если у вас есть логотип, создайте иконки используя любой графический редактор:
- Photoshop
- GIMP
- Figma
- Canva

Сохраните каждый размер как `icon-{размер}.png` в папке `public/images/icons/`

## Вариант 3: Простое решение для тестирования

Создайте простую иконку с текстом "О+" на цветном фоне.

Все иконки должны быть в формате PNG с прозрачным фоном (или с цветным фоном #2563eb - основной цвет проекта).

## Для iOS

Дополнительно создайте файл `apple-touch-icon.png` размером 180x180px для отображения на главном экране iOS устройств.
